AI Study Assistant (RAG)

PythonLangChainChromaDB StreamlitGroqSentenceTransformersDocker
Jun 2026

An AI-powered study assistant that allows students to upload learning materials and ask questions about their documents using natural language. The system uses Retrieval-Augmented Generation (RAG) to retrieve relevant information from uploaded files before generating responses, reducing hallucinations and improving answer accuracy. Supports multiple document formats, provides source citations with every response, and runs locally using vector embeddings stored in ChromaDB.

Key Features
Retrieval-Augmented Generation Multi-Document Q&A Source Citation Tracking Local Vector Database Streamlit Interface Dockerized Deployment
